Mix Prep / Editing Prep

Track Organization

  • To start, import the mix template that provides a few utilities

    Then import all the audio and make sure all tracks are descriptively labeled. "Kick in"/"K", "Rm", "Long Verb", "LngVb", "Sn Plate","Ld Gtr" ,etc., are perfectly acceptable.

    If we were given a PTX, at minimum let's import the audio tracks from the PTX. If the routing and organization of the original PTX is good, we can use some of their busses. This will have to be case-by-case.

  • Bonus: use the comments box to write any committed processing, or information I need to know about the content of playlists.
  • Another bonus: please put multiple takes of a source in a playlist. Some projects do not come in as PTX and each take is just a wav file from a big mess in a folder.
  • Color coding:
    Please use these brightness levels out of Pro Tools 3 vertical rows:
    • Busses and Ld Vocals: Top row (brightest)
    • Other audio source tracks: Middle row
    • FX, Delays, Triggers: Bottom row (darkest)
    Please follow this color coding and place the tracks in this general order:
    • Drums: Red
    • Drum Bus goes here. Brightest Red.
    • Bass: Orange
    • Guitars: Green
    • If both acoustic guitars and electric guitars, push electric guitars towards a yellow-green and the acoustics towards lime green.
    • Synths: Sky Blue
    • Horns: Yellow
    • Band Bus goes here. I like bright blue.
    • Vocals: Pink for leads, purple for harmonies. Use top row pink for lead despite following notes about color brightness. Lead doubles should be middle row of the same pink.
    • Vocal FX go here. Darkest purple.
    • Vocal Bus goes here. Bright purple.

Bussing

  • Drums go to the drum bus. If its a live drum kit with phase-relative muliple mics, please put those mics in an edit group. No VCA needed in this case.
  • Multi-mic'd sources or more than 4 channels per instrument should go to their own (edit/mix if phase locked) mix group with a VCA fader just for organization sake. Examples are a pile of rhythm guitar tracks by the same performer getting a mix group with VCA, multiple mics on a lead guitar solo getting an edit/mix group with VCA, a bynch of synths getting a mix group with VCA.
  • Everything that isn't a vocal or vocal effect goes to the BAND bus.
  • Everything that is a vocal or vocal related should go to the VOCAL bus.
  • Send the ld vocals to all 5 vocal effect auxes. Don't worry about backing vocals.
  • Only the BAND and VOCAL busses should go to the MIX bus. Everything should be in one of those two busses before hitting the MIX.
  • I like to have busses after the tracks feeding them. So general track order would be a pile of drum tracks, then DRUM BUS, then a pile of instruments, BAND BUS, a pile of vocals, vocal effects, VOCAL BUS, MIX BUS, MONITOR MASTER

Editing:

  • First, make a new or duplicate playlist. Leave all the source material in playlists. Sometimes I like to reference the unedited material to handle a mix note.
  • Comp and time edit as much as the song requires. Some genres lend themselves well to lots of comping, some do not.
  • Please use RX De-noise, De-click, vocal de-noise, etc., to repair just the trouble spots through audiosuite, commiting the plugin.
  • Melodyne as much as required. Please commit melodyne to a new playlist.

Last steps:

  • Drop all audio track faders to -inf (silent).
  • Export the PTX as YYMMDD Client Name - Project Name. Some exceptions apply. I will let you know if so.
  • I will provide cloud space to deliver PTXs.
